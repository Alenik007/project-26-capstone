# AI Interview Coach

## Проблема

Кандидатам сложно готовиться к собеседованиям под конкретную вакансию: они читают общие списки вопросов, которые часто не совпадают с реальными требованиями работодателя.

## Целевой пользователь

Junior / Middle IT-специалист, который готовится к собеседованию на позицию Python Developer, Backend Developer, Data Engineer, DevOps Engineer, Data Analyst или System Analyst.

## Как пользователь решает задачу сейчас

- вручную читает вакансию;
- ищет вопросы в интернете;
- просит ChatGPT придумать вопросы;
- сам оценивает свои ответы;
- не получает структурированную обратную связь.

## Решение

AI-агент анализирует конкретную вакансию, генерирует персонализированные вопросы, проводит mock-интервью и даёт обратную связь.

## Стек

- LangGraph для автономного агента;
- OpenAI API для LLM;
- FastAPI для backend;
- Next.js для frontend;
- Qdrant для RAG;
- Docker для упаковки;
- AWS EC2 Ubuntu для деплоя;
- Nginx для reverse proxy.

## Почему выбран этот стек

LangGraph подходит для автономных агентов с инструментами. FastAPI удобен для streaming API. Next.js позволяет быстро сделать frontend. Qdrant подходит для RAG-поиска по базе знаний. AWS EC2 даёт полный контроль над сервером и Docker Compose деплоем.
